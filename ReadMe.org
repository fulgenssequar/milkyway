* rules.js:

  Explaination of the transforming rules:
  
** -s
   meets -> meet
   
** -g-n-i+e
   voting -> vote

** -s-e-i+y
   treaties -> treaty
   
** -g-n-i-%2-%s-%1$$
   sitting -> sit
   
- %1 = aeiou
- %2 = bcdfghjklmnpqrstvwxyz
- %s = same charactor
- $ = pop
